# Product Manager Agent

Wir brauchen einen weiteren optionalen agent der ausgeführt wird, wenn noch nciht klar ist wie das feature aussehen soll.

Zunächst soll über ein flag "--product_manager" dieser neue agent aktiviert werden und ncoh vor dem architekten angeschmissen werden.

Aufgabe des Agents:
- Business Case des neuen Features verstehen
- Unklare Anforderungen klären
- Fachliche bewertung des features, also wie es in die Anwendung integriert werden soll
- Wenn es thematisch passt alternatvien vorschlagen

Tools:
- research
- ui design (kann auch schon ein design erstellen, dann muss es der Architekt nicht machen)
